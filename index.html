Selamat datang di website profil sederhana saya. Website ini dibuat untuk latihan HTML & CSS dasar.

Situs ini berisi biodata, hobi, dan cita-cita saya sebagai siswa HSI Boarding School Purworejo. Silakan jelajahi tautan navigasi di atas untuk mengenal saya lebih dekat.

Tentang Portal Ini

Portal ini dirancang dengan pendekatan mobile-first, menggunakan Flexbox untuk layout responsif di semua perangkat (mobile, tablet, desktop). Navigasi di menu, eksplorasi galeri foto, lihat jadwal kegiatan, dan kirim feedback dengan mudah!
